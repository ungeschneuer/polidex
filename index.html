Was ist das?

PANINI-ALBUM FÜR DAS PARLAMENT.

Du öffnest einen Nachrichtenartikel. Du klickst „Jetzt scannen!". Das Plugin durchsucht den Text nach Bundestagsabgeordneten, präsentiert Treffer als glänzende Sammelsticker — und du klebst sie mit einem Klick ins Album.

Jede*r Politiker*in lässt sich nur einmal sammeln. Wer schon im Album klebt, bekommt bei jeder weiteren Erwähnung XP. Level steigt mit √XP. Tagesserien geben Bonus.

Das Ziel: Alle 1.865 Bundestagsabgeordnete*n der Wahlperioden 16–21 ins Heft.

Anleitung

SO GEHT’S.

  1. 01

    Artikel öffnen

    Egal welche Nachrichtenseite — spiegel.de, tagesschau.de, taz.de, lokales Blatt.

  2. 02

    Plugin klicken

    Roter Knopf: „Jetzt scannen!". Das Plugin liest den Artikel — nichts verlässt den Browser.

  3. 03

    Druff!

    Pro Treffer ein Sticker. Klick auf „+ Druff!" und schon klebt’s im Album.

  4. 04

    Sammeln & leveln

    Jede weitere Erwähnung gibt XP. Achievements freischalten, Serie aufbauen, alle 1.865 sammeln.

Dein Album

Sortier nach Partei, Level, Wahlperiode oder „wer fehlt noch?". Jede weitere Erwähnung gibt XP — Abgeordnete mit langer Parlamentserfahrung sind Legendär.

Privatsphäre

KEINE DATEN.
KEIN BACKEND.
KEIN BLA.

  • ✓
    Alles bleibt lokal. Dein Album lebt in browser.storage.local. Nichts verlässt deinen Browser.
  • ✓
    Kein Konto, kein Server. Politikerdaten kommen direkt von abgeordnetenwatch.de und Wikidata — beide öffentlich, kein Tracking.
  • ✓
    Open Source. Vollständiger Code auf GitHub. MIT-Lizenz. Audit it yourself.
  • ✓
    Nur, was gelesen wird. Das Plugin liest Artikeltext nur, wenn du auf „Jetzt scannen!" klickst. Kein Hintergrundgescanne. Kein Profil.
