Willkommen!
Album ist bereit, Klebstoff aufgewärmt.

Hier siehst du, wie das Ganze funktioniert — und wie du das Plugin in zwei Klicks an die Symbolleiste pinnst.

Anleitung

SO GEHT’S.

  1. 01

    Artikel öffnen

    Jede Nachrichtenseite tut’s — spiegel.de, tagesschau.de, taz.de, dein Lokalblatt.

  2. 02

    Plugin klicken

    Oben rechts im Browser auf das gelbe Icon klicken. Dann auf den roten Knopf „Jetzt scannen!".

  3. 03

    Druff!

    Für jede erkannte Person ein Sticker. Klick auf „+ Druff!" klebt ihn ins Album.

  4. 04

    Sammeln & leveln

    Jede weitere Erwähnung gibt XP. Achievements freischalten, Serie aufbauen, alle 1.865 jagen.

Empfohlen

AN DIE LEISTE PINNEN.

Damit du das Plugin mit einem Klick erreichst — sonst versteckt es sich im Erweiterungs-Menü.

CHROME Edge · Brave · Arc · Vivaldi
  1. Klick auf das Puzzle-Symbol rechts neben der Adressleiste.
  2. Suche Politi-Sammelalbum in der Liste.
  3. Klick auf das Pin-Symbol daneben.

✓ Erscheint jetzt dauerhaft in der Symbolleiste.

FIREFOX ab Version 128
  1. Klick auf das Erweiterungs-Symbol rechts neben der Adressleiste.
  2. Suche Politi-Sammelalbum in der Liste.
  3. Klick auf das Zahnrad und wähle „An Symbolleiste anpinnen".

✓ Erscheint jetzt dauerhaft in der Symbolleiste.

Hinweise

GUT ZU WISSEN.

  • ✓ Alles bleibt lokal. Kein Server, kein Konto, kein Tracking. Dein Album lebt nur in deinem Browser.
  • ✓ 24 Achievements warten — von „Erstabdruck" über „Plenarsaal-Mobiliar" bis „Maximum Talkshow".
  • ✓ Export & Import als JSON unter Sammlung → Export. Backup oder Geräte­wechsel: kein Problem.
  • ✓ Partei- und Regierungs-Seiten werden nicht gescannt (z. B. spd.de, bundestag.de) — die zählen nicht als Nachrichten.
  • ✓ Hilfe jederzeit: Im Popup oben rechts auf ? für die Spielregeln. Oder Esc drücken, um Karten zu schließen.

Diese Seite öffnet sich nur einmal. Über das ? im Popup kommst du jederzeit zurück zu den Spielregeln.
