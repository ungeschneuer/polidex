Einleitung

Was ist Polidex?

Polidex ist eine Browser-Extension für Chrome und Firefox. Öffne einen Nachrichtenartikel, klicke auf „Seite scannen“, und Polidex erkennt automatisch, welche Bundestagsabgeordneten im Text erwähnt werden. Sammle sie alle, von den Dauerbrennern bis zu den ganz seltenen Hinterbänkler*innen!

Alle Daten bleiben ausschließlich lokal in deinem Browser — kein Backend, kein Konto, kein Upload zu Drittservern. Die Politiker*innendaten werden täglich von abgeordnetenwatch.de und Wikipedia synchronisiert.

Ziel: Alle 1.865 Abgeordnete*n der Wahlperioden 16–21 (2005–heute) sammeln — vom aktuellen 21. Bundestag (630 Abgeordnete) bis zurück zum 16. Bundestag (2005–09).

Anleitung

So funktioniert’s

01
// ARTIKEL ÖFFNEN

Artikel öffnen

Öffne einen Nachrichtenartikel auf einer beliebigen Nachrichtenseite.

02
// SEITE SCANNEN

Seite scannen

Klicke auf „Seite scannen“ im Polidex-Popup oben rechts im Browser.

03
// SAMMELN

Sammeln

Erkannte Politiker*innen mit einem Klick zur persönlichen Kollektion hinzufügen.

04
// AUFLEVELN

Aufleveln

Scanne weitere Artikel, um XP zu sammeln, Level zu steigen und Errungenschaften freizuschalten.

Datenschutz

  • ✓ Alle Daten werden ausschließlich lokal in deinem Browser gespeichert – nichts verlässt dein Gerät.

  • ✓ Open Source — der vollständige Quellcode ist auf GitHub einsehbar.
